Nautilus 2K16

This is tech fest web app which done in association with college tech fest nautilus2k16 . It is rails based app designed to help manage tech events.

Features

  • Based on Rails Platform
  • Easy User Ticket Mangament
  • Rails Admin
  • Android API
  • Material Design

Installation

  1. Clone Repository
  2. Run

    bundle Install

  3. Run Server

    rails s

  4. Add Events usings Rails-admin

    /admin

Rails Gems

  • Devise - User Auth
  • Rails-Admin - Admin Panel
  • Puma - Server
  • Capistrano - Deployment
